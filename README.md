# yukinyaa.github.io

Portal page for Yukinyaa GitHub Pages site.

## Local preview

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Deployment

Changes pushed to `main` are deployed to the `gh-pages` branch by GitHub Actions.
